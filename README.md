난달이란 길이 여러 갈래로 통한 곳이라는 순우리말로,  
사통팔달로 된 곳이 교통의 중심지가 되어 사람들이 모여들어 번화한 것처럼  
여행 가기 전 필수적으로 거쳐야 하는 사이트를 지향하고 있다.  
정보에 대한 접근성을 높여 이용자에게는 취향에 맞는 관광 정보를 제공하고  
지역사회에는 관광 정보에 대한 접근성을 높여 지역발전에 기회를 높인다.
